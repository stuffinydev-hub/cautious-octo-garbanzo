HAKOGRAM - SETUP INSTRUCTIONS
=============================

1. CREATE GITHUB REPOSITORY
   - Go to https://github.com/new
   - Name: hakogram-ios
   - Description: Hakogram - Enhanced Telegram iOS Client
   - Public or Private (your choice)
   - DO NOT initialize with README

2. SETUP GITHUB SECRETS
   Go to: Repository Settings → Secrets and variables → Actions
   
   Add these secrets:
   - TELEGRAM_API_ID: Your Telegram API ID from https://my.telegram.org
   - TELEGRAM_API_HASH: Your Telegram API Hash
   - APPLE_TEAM_ID: Your Apple Developer Team ID

3. PUSH TO GITHUB
   Open terminal in ghostgram folder and run:

   git init
   git add .
   git commit -m "Initial commit - Hakogram iOS"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/hakogram-ios.git
   git push -u origin main

4. GITHUB ACTIONS WILL AUTO-BUILD
   - Go to Actions tab in your repository
   - Build will start automatically
   - Download IPA from Artifacts after build completes

5. ADMIN PANEL SETUP
   - Edit: jutsogram/AdminPanel/AdminPanelManager.swift
   - Line 9: Set ADMIN_USER_ID to your Telegram user ID
   - Line 10: Set ADMIN_API_ENDPOINT to your backend URL (optional)

6. GET YOUR TELEGRAM USER ID
   - Message @userinfobot on Telegram
   - It will reply with your user ID
   - Use this number in AdminPanelManager.swift

7. FEATURES ENABLED
   ✓ JutsoAI - AI Chat with 10 models
   ✓ Virtual Stars & NFT Gifts
   ✓ Ghost Mode - Privacy features
   ✓ Anti-Delete - Recover deleted messages
   ✓ Voice Morpher - Voice effects
   ✓ Send Delay - Delay messages
   ✓ Admin Panel - Full control over users

8. ADMIN PANEL FUNCTIONS
   - View all Hakogram users
   - Ban/Unban users
   - Grant stars to users
   - Send broadcast messages
   - View user statistics
   - Disable features for specific users
   - View user logs
   - Force updates

9. CONTACT
   Telegram: https://t.me/jutsodev

IMPORTANT NOTES:
- Keep your API credentials secret
- Never commit build-configuration.json
- Admin Panel only works for you (ADMIN_USER_ID)
- Users are tracked automatically when they use Hakogram
